# Crime Data Analysis Portal

## Description
The Crime Data Analysis Portal is a web-based analytics application designed to visualize and analyze crime data using interactive dashboards.

The project integrates Tableau Public dashboards with a custom frontend built using HTML, CSS, and JavaScript. The dashboard is displayed only when required to maintain a clean and user-friendly interface.

The dataset used in this project is sourced from Kaggle and contains structured crime records across multiple regions and time periods.

## Technologies Used
- HTML
- CSS
- JavaScript
- Tableau Public
- GitHub Pages

## Deployment
The project is deployed using GitHub Pages and is publicly accessible.
